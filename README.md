# Wellness & Sports Dashboard

## Overview

This project analyzes employee sports activity and its impact on HR benefits (bonuses, well-being days).

## Tools

* Tableau Public
* CSV datasets

## Key Metrics

* Number of employees
* Eligible employees (bonus & well-being)
* Total cost of bonuses
* Activity level distribution

## Insights (WIP)

* Higher activity levels correlate with eligibility
* Distribution of activities is skewed toward mid/high levels
